# 1 安装显示字体  
## 1.1 字体管理
使用**fontconfig**工具集管理字体配置,缓存和渲染    
  - 显示所以已安装字体  
  `$ fc-list`  
  - 更新字体缓存  
  `$ fc-cache -fv`
    - "-f": 强制更新缓存
    - "-v": 显示详细过程
  
存放字体的路径:
  - 系统级字体存放: /usr/share/fonts/
  - 用户级字体存放: ~/.local/share/fonts/	  


## 1.2 安装字体
使用**pacman**命令通安装字体   
`$ sudo pacman -S noto-fonts noto-fonts-cjk noto-fonts-emoji ttf-jetbrains-mono`
  
将字体文件存放到字体目录     
  - 将字体文件复制到字体目录,根据存放目录的不同可能需要root权限    
  - 更新字体缓存


## 1.3 卸载字体
使用**packman**命令卸载字体
`sudo pacman -Rs noto-fonts`

将字体文件从字体目录删除
  - 将字体文件从目录中删除,根据存放目录的不同可能需要root权限
  - 更新字体缓存   

> 存放字体的路径:
>   - 系统级字体存放: /usr/share/fonts/ (需要root权限)
>   - 用户级字体存放: ~/.local/share/fonts/ (不需要root权限)


      
# 2 安装桌面环境相关软件  
  1. 窗口管理器：sway  
  2. 任务状态栏：swaybar  
  3. 状态栏生成器：i3blocks  
  4. 终端模拟器：foot  
  5. 应用启动器：wmenu  


  
# 3 sway配置
进入用户的sway配置文件,进行以下配置: 
  1. 默认键设置: mod键与方向键  
  2. 默认软件设置: 默认终端模拟器和应用启动器
  3. 显示设置: 整体字体设置,分辨率和缩放设置
  4. 按键设置: sway功能按键设置,其他工具按键设置(声音,亮度,截屏)
  5. 状态栏设置: 设置**i3blocks**软件为状态栏生成器,设置状态栏的外观(位置,高度,字体)
  6. sway启动设置: 设置**fcitx5**软件随sway一起启动 


  
# 4 i3blocks配置
进入用户的i3blocks配置文件,进行以下配置:  
  1. 配置公共属性: 分离字符设置,分离距离设置,文本处理库
  2. 编写各状态栏模块: 包括调用的命令脚本,调用的频率,触发调用的信号

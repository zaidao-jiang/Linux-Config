# 1 下载相关功能软件
yazi的功能需要额外下载其他软件:
  - 预览功能  
    - "7zip": 预览压缩文件  
  - 搜索功能  
    - "fd": 搜索文件名称  
    - "ripgrep": 搜索文件内容  
  - 跳转功能  
    - "fzf": 形成文件目录树并模糊搜索 
    - "zoxide": 形成历史目录记录



# 2 yazi配置
通过bash配置文件(~/.bashrc):
  - 当退出yazi时更改工作目录到退出时目录并且使用“y”代替“yazi”  
    `function y() {
             local tmp="$(mktemp -t "yazi-cwd.XXXXXX")" cwd
             command yazi "$@" --cwd-file="$tmp"
             IFS= read -r -d '' cwd < "$tmp"
             [ -n "$cwd" ] && [ "$cwd" != "$PWD" ] && builtin cd -- "$cwd"
             rm -f -- "$tmp"
    }`

从github从获取默认配置文件到用户配置文件夹(~/.config/yazi):
  - "yazi.toml": 功能配置,预览配置,打开文件的软件配置,流程解释配置
  - "theme.toml": 颜色,图表,大小配置
  - "keymap.toml": 按键配置

> github配置文件地址：https://github.com/sxyazi/yazi/tree/shipped/yazi-config/preset

# 4 yazi使用
## 4.1 查看文件
  - 目录移动
  - 查看文件简单信息("tab")
  - 显示文件多种信息("m")
  - 复制文件路径或名称("c")
  - 打开文件
    - "o": 直接使用默认软件打开
    - "O": 出现可供选择的软件


## 4.2 修改文件  
  - 文件重命名("r")
  - 创建文件("a")
    - "filename.extension": 创建文件
    - "filename/": 创建文件夹
  - 删除文件
    - "d": 将文件移除到系统回收站
    - "D": 永久强制删除文件


## 4.3 移动文件  
  - 选择与取消文件("space")
  - 复制文件("y")
  - 取消复制文件("Y")
  - 剪切文件("x")
  - 取消剪切文件("X")
  - 粘贴文件("p")
  - 强制粘贴并覆盖相同命名文件("P")
  

## 4.4 管理多文件
  - 排序文件(",")  
  - 筛选文件("f"): 筛选当前目录文件 
  - 搜索文件  
    - “s”：递归搜索当前目录文件  
    - “S”：递归搜索当前目录文件的内容  
  - 跳转文件  
    - “z”：模糊递归搜索当前目录并排序  
    - “Z”：根据历史形成目录记录  
    - “g”：跳转已经列出的路径,可以在keymap.toml配置文件中增加路径  


## 4.5 其他功能
  - 标签页
    - "t": 新建标签页
    - "ctrl+c": 删除当前标签页
    - "1,2,3,4,5,...": 切换不同标签页
  - 显示运行任务("w")

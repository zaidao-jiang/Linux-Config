# 2 fcitx5
安装相关软件:  
`$ sudo pacman -S fcitx5 fcitx5-im fcitx5-chinese-addons`  

输入法配置:  
  1. 进入GUI配置界面  
  `$ fcitx5-configtool`  
  2. 添加中文输入法: 进入“Input config”页面,从右边栏搜索“pingyin”并添加到左侧.
  3. 设置语言切换按键: 进入“Global Setting”页面,"Trigger Input Method"选择"Right Shift".
  4. 禁用临时语言切换案件: 进入“Global Setting”页面,"Temporally switch between first and current Input Method"选择为"empty".
  5. 设置默认主题: 进入“Addons”页面,进入"Classic User Interface"设置"Theme"为"DefaultDark".  
  5. 将fcitx5当做守护进程运行,已在sway文件配置

# 7 steam  
下载32位硬件驱动:
  1. 启动multilib库,编辑"/etc/pacman.conf"文件,取消[multilib]段落
  `$ sudo helix /etc/pacman.conf`
  2. 更新当前库  
  `$ sudo pacman -Syy`  
  3. 安装32位OpenGL和Vulkan的AMD和Nvidia驱动  
  `$ sudo pacman -S lib32-mesa lib32-vulkan-radeon lib32-vulkan-nouveau`

安装sway下兼容xorg的软件
`$ sudo pacman -S xorg-xwayland`  

在steam设置里找到兼容性，选择Proton的最新软件 

> 可使用www.protondb.com查看游戏的兼容情况

# 5 使用帮助文档  
使用**man**命令查询命令或软件的帮助文档.
可以先通过模糊搜索获取所有和命令或软件相关的内容,再根据搜索的内容进行精确查询:  
  1. 生成关键词缓存  
  `$ sudo mandb`  
  2. 模糊搜索内容  
  `$ man -k 搜索内容`
  3. 精确查询内容
  `$ man 查询内容`



# 0 配置文件管理
使用**stow**软件,通过管理符号链接来管理软件包或用户配置文件,
将软件包或文件集中存储一个目录,通过符号链接给目标目录,便于集中管理分散的软件或配置文件
  - "stow directory": 存放软件和配置文件的根目录
  - "stow packagename directory": 不同软件和配置文件在"stow directory"下创建的各自子目录
  - "target directory": 存放符号链接的目录,默认是"stow directory"的父目录

使用**stow**命令创建符号链接时,将"stow packagename direcory"下的内容,按照层级结构,在"target dirctory"下对应创建符号链接
`stow packagename`: 在默认目标目录处创建符号链接
`stow -t target directory packagename`: 在指定的目标目录处创建符号链接

使用**stow**命令删除符号链接
`stow -D packagename`

使用**stow**命令重新链接(先删除再创建,用于更新)
`stow -R packagename`
